Domande Frequenti (Faq)

1. Cos'è I'm Powah - Trova Primi ?
2. Quali sono i requisiti di sistema ?
Risposte

1. I'm Powah - Trova Primi è un programma che si avvia in DOS che, dato un numero qualunque, crea nella stessa cartella dove si trova, un file di testo numeriprimi.txt dove vi sono tutti i numeri primi dopo il numero immesso.
Il software è capace di trovare miliardi di numeri in poco tempo ed inoltre è utile anche per creare file di centinaia di megabyte...

2. I'm Powah - Trova Primi richiede solamente di girare in un Sistema Operativo Microsoft Windows XP, Vista o 7. Le prestazioni dipendono dal processore in uso.
